# supermap

> SuperMap iClient for JavaScript 9D


## Features

* 云GIS网络客户端开发平台
* 基于`现代Web技术栈`全新构建
* `集成`了领先开源地图库、可视化库，且核心代码以Apache2协议完全开源
* 提供了一下`5`个Libraries，支持2d和3d地图引擎：
    * iClient for Leaflet
    * iClient for OpenLayers
    * iClient for MapboxGL
    * iClient Classic
    * iClient 3D-WebGL
