# url-schema

> 浏览器或webview中通过通用链接方式打开APP

## Schemas

    ucbrowser://                    # UC浏览器            
    mttbrowser://                   # QQ浏览器            
    weixin://                       # 微信                
    baiduboxapp://                  # 百度APP             
    baiduboxapp://v11/test/123

* 在原生浏览器中一般能正常使用
* 在微信中基本不好使了，估计得添加域名白名单才可
