## 常用命令

    $ emacs

    C-z                 挂起Emacs
    C-x C-z             退出Emacs

    C-x C-f             打开文件
    C-x C-s             保存文件
    C-x s               保存所有文件
    C-x i               插入其他文件到当前缓冲区
    C-x C-v             在当前缓冲区打开另一个文件
    C-x C-W             另存为文件
