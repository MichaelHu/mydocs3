# neovim

## Features

* `Neovim` is the future of Vim - 是vim的未来
* 基于vim项目进行重构，提升其维护性，并实现了很多新的高级特性
* 提供更好更强大的`插件系统`
* 提供现代化的GUI
* 对其他语言提供API调用 
* 内嵌可编程的终端模拟器
* 异步任务控制
* 多个编辑器实例之间共享数据 - `shada`
* 基于`XDG`的目录支持
* 兼容大部分的vim插件，包括Ruby和Python插件
