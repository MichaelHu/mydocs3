# es
> Elastic Search

* 基于**Lucene**的开源搜索引擎，在Java开发环境中，Lucene是一个成熟的免费开源工具，它是一个全文检索引擎，提供完整的**查询引擎**和**索引引擎**。
* ES是一个**实时分布式搜索和分析引擎**，它让你以前所未有的速度处理大数据成为可能
* ES的一些使用案例如下：
    1. **维基百科**使用ES提供全文搜索并高亮关键字，以及输入实时搜索（search-asyou-type）和搜索纠错（did-you-mean）等搜索建议功能
    2. **英国卫报**使用ES结合用户日志和社交网络数据提供给他们的编辑以实时的反馈，以便及时了解公众对新发表的文章的回应
    3. **StackOverflow**结合全文搜索与地理位置查询，以及more-like-this功能来找到相关问题和答案
    4. **Github**使用ES检索1300亿+行代码
