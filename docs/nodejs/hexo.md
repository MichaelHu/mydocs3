# hexo

> A fast, simple & powerful blog framework.


## Features

* 快速生成blog
* github风格markdown语法
* 一键部署至github pages， heroku等 
* 强大插件系统

## Usage

    $ npm install hexo-cli -g
    $ hexo init blog
    $ cd blog
    $ hexo server
    $ hexo new "Hello Hexo"
    $ hexo generate
