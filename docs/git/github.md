# github

## 修改项目语言

* 默认使用`数量`最多的文件类型作为语言类型
* 通过修改`.gitattributes`文件，可以强制更改

修改方式如下：

    $ vim .gitattributes
    *.js linguist-language=Node
    *.css linguist-language=Node
    *.html linguist-language=Node
