# jsplumb

> Build `Connectivity` Fast.


## Features

* 使用`svg`技术进行渲染，无第三方图形库依赖
* 主要针对流程图、关系图等基于`点、线关系`的图的`可视化`
* 比较适合制作`流程图`、`UML图`等，站点首页为两个Demo
