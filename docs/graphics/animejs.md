# animejs

> Anime.js (/ˈæn.ə.meɪ/) is a lightweight JavaScript animation library with a simple, yet powerful API. It works with CSS properties, SVG, DOM attributes and JavaScript Objects.

## Features

* 轻量级动画引擎
* 使用CSS，SVG，DOM等来实现
