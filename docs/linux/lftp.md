# lftp

## Features

* a sophisticated file transfer program
* supporting ftp, http, sftp, fish, torrent


## Resources

* site: <http://lftp.yar.ru>
* `sftp`: <ref://./sftp.md.html>


## Install

    # mac
    $ brew install lftp


## Usage

    $ lftp <user>:<password>@host
    $ help
    $ pwd
    $ cd ...
    $ lpwd
    $ lcd ...
    $ put ...
    $ get ...
    $ bye
