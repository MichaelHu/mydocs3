## Managers
> 包管理器**因平台而异**

| 操作系统  | 格式          | 工具                          |
| Debian    | .deb          | apt, apt-cache, apt-get, dpkg |
| Ubuntu    | .deb          | apt, apt-cache, apt-get, dpkg |
| CentOS    | .rpm          | yum                           |
| Fedora    | .rpm          | dnf                           |
| FreeBSD   | Ports, .txz   | make, pkg                     |
