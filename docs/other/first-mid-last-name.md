# first-mid-last name

## First Name

名

## Last Name

姓

## Middle Name

* 不是中文名字中间的那个
* 欧美国家信仰基督教或天主教的人们起的教名

## Example

### George W. Bush

* First Name ( 名 ): George
* Middle Name ( 教名 ) : W. ，这里是教名的缩写
* Last Name ( 姓 ) : Bush


### 胡大民 

* First Name ( 名 ): 大民
* Middle Name ( 教名 ) : 无 
* Last Name ( 姓 ) : 胡
