# hhkb
> Happy Hacking KeyBoard

## Overview
* 程序员键盘，共`60键`，vim使用者福音
* PFU公司产品，一家属于Fujitsu（富士通）的公司，官方站点：<https://happyhackingkb.com>


## 使用提示
> 大部分配置，可以在键盘背面的说明上找到，以下信息在说明书上才有，摘出如下：
* 删除个别配对信息
    1. 第一步：`Fn + Q`
    2. 第二步：`Fn + Ctrl + Delete`后，再按下1-4，选择要删除哪个配对
* 一次性删除所有配对信息 
    1. 第一步：`Fn + Q`
    2. 第二步：`Fn + Z + Delete`
