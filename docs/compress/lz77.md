# lz77

## Features

* `基于字典`的、`“滑动窗”`的`无损`压缩算法
* `Lempei-Ziv`于`1977`年发明
* `GBA`内置的压缩算法
* 严格意义来讲，LZ77不是一种算法，而是一种`编码理论`，同Huffman编码一样，只定义了原理，并没有定义如何实现
* 影响了当今主流压缩工具的实现，比如ARJ, PKZip, WinZip, LHArc, GZip, ACE, ZOO, TurboZip, Compress, JAR..
