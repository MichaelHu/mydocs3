# catmull-rom

> 获得经过控制点的平滑曲线

## Overview

 <img src="./img/catmull-rom_spline.png">

四个控制点P0, P1, P2, P3，获得`连接P1, P2`的`光滑曲线`。常见应用于`曲线差值`。
