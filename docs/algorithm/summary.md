# algorithm summary


> 直白地说，算法就是任何明确定义的计算过程，它接收一些值或集合作为输入，并产生一
> 些值或集合作为输出。这样，算法就是将输入转换为输出的一系列计算过程。来源：
> Thomas H. Cormen, Chales E. Leiserson (2009), 《算法导论第三版》。
