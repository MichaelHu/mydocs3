# bootstrap-table

## Features

* 表头固定
* 列排序
* 响应式布局
* 支持多格式下载
* ...

## Install

    $ bower install bootstrap-table
    $ npm install bootstrap-table


## Tips

* 最新版依赖jQuery版本，1.9.x版本不支持，**建议使用3.x版本**
